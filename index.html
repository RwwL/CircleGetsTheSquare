Simple Background Art Generator

Takes an element, creates and prepends an HTML5 canvas element (if supported), and draws a series of randomized circles and rotated rectangles on the canvas based on the following default options, or your custom options:

artGen.defaultOptions = {

	regenOnWinResize	: true,
	canvasClassName		: 'artGenCanvas',
	
	// if set, overrides CSS color on element
	// (set CSS value for non-canvas browsers)
	bgColor			: 'white',

	// randomized circle radii
	minRadius		: 30,
	maxRadius		: 48,

	// randomized dimensions
	minRectSide		: 60,
	maxRectSide		: 150,

	// randomized color values
	// (generates rgba)
	rMin			: 0,
	rMax			: 255,
	gMin			: 0,
	gMax			: 255,
	bMin			: 0,
	bMax			: 255,
	aMin			: 0.2,
	aMax			: 1,

	// x/y spacing for center points
	spacing			: 80,

	// randomized +/- variance of center points
	// to avoid appearance of a rigid grid
	jitter			: 8,
};

Presently requires jQuery for some convenience with DOM manipulation and dimension-getting, so I wrapped it up as a plugin as well (but it could also just be called with artGen.generate(element, options).

Typically takes between 30-80ms to run.

Will regenerate on window.resize unless you set the relevant option to false.
